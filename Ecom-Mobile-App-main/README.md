# Luxe Ecom - Golden Midnight Edition 📱✨

A high-end, premium E-Commerce mobile application reimagined with a sophisticated **Golden Midnight** aesthetic. Built with **React Native** and **Expo SDK 55**, this app combines artisanal design with fluid performance.

## ✨ Premium Features

- **🌙 Golden Midnight Theme**: A sophisticated dark aesthetic with metallic gold accents for a luxury feel.
- **🔐 Elegant Auth**: Polished Login and Signup flows with high-contrast inputs.
- **🏠 Cinematic Home**: Modern banner layouts and featured discovery sections.
- **🔍 Refined Discovery**: Intuitive search and category exploration with premium card designs.
- **📄 High-End Product View**: Immersive details, cinematic imagery, and glassmorphism-inspired UI elements.
- **🛒 Seamless Cart**: Intuitive quantity management with a refined checkout preparation flow.
- **❤️ Curated Wishlist**: Save your luxury favorites in a dedicated collection view.
- **👤 Elite Profile**: Minimalist user management and unified settings.
- **📦 Order History**: Track past investments with clear status indicators.
- **⚡ Fluid Navigation**: Whisper-smooth transitions using Native Stack and Bottom Tabs.

## 🚀 Tech Stack

- **Framework**: [Expo](https://expo.dev/) (SDK 55)
- **Library**: [React Native](https://reactnative.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Navigation**: [React Navigation](https://reactnavigation.org/)
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)
- **Icons**: [Expo Vector Icons](https://docs.expo.dev/guides/icons/)
- **Safe Area**: [React Native Safe Area Context](https://github.com/th3rdwave/react-native-safe-area-context)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS)
- [Expo Go](https://expo.dev/client) app installed on your physical device.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/saurabhxcod/Ecom-Mobile-App.git
   cd Ecom-Mobile-App
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npx expo start
   ```

### Running the App

- **Android**: Scan the QR code using the **Expo Go** app.
- **iOS**: Scan the QR code using the **Camera** app.
- **Emulator/Simulator**: Press `a` for Android or `i` for iOS in the terminal.
